# 介護コミュニケーションツール (kaigo-tool)

言語障害を抱える方が、クリック操作によって自身の「感情」や「緊急性」を相手に伝えるための介護支援Webアプリケーションです。

## 公開サイト
- [GitHub Pagesでアプリを動かす](https://github.io)

## 開発背景と目的
- **課題意識**：言語障害を抱える祖母が、会話も文字入力もできず、手だけの表現では細かい感情を伝えられずにストレスを抱えている現状に直面しました。
- **解決策**：単なる文字読み上げではなく、**「クリックの速度に応じて、読み上げの速度と音量が変化する仕組み」**を考案。これにより、言葉が発せなくても本人の感情、重要性、緊急性を豊かに表現し、周囲へ正確に伝えられるようにしました。

## 主な5つの機能
1. **感情・緊急性を伝える読み上げ機能**（速度・音量を動的に変化）
2. **毎日の健康セルフチェック機能**（日々の体調を簡単に記録）
3. **日記・記録機能**（日々の出来事や気づきを保存）
4. **ToDoリスト機能**（タスクや申し送りを管理）
5. **Firebase認証機能**（Googleアカウントによる安全なログイン）

## 使用技術
- **フロントエンド**：HTML5 / CSS3 / JavaScript
- **バックエンド/認証**：Firebase (Google Authentication)
- **デプロイ**：GitHub Pages
